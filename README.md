
# 🎓 CGPA Calculator (Python + Tkinter)

A simple **CGPA Calculator** built using **Python Tkinter** that allows students to calculate their GPA, grades, and overall CGPA easily.  
The application provides a user-friendly interface where you can dynamically add subjects, input marks, and instantly view results along with the grading system.

---

## ✨ Features
- ➕ **Add Subjects Dynamically** – no fixed subject limit.  
- 🧮 **Automatic GPA & Grade Calculation** based on marks.  
- 📊 **CGPA Calculation** with two decimal precision.  
- 📖 **Grading System Reference** displayed with results.  
- 🎨 **Simple & Intuitive GUI** built using Tkinter.  
- 🚫 **Error Handling** for invalid inputs (e.g., empty fields or non-numeric entries).  

---

## 📚 Grading System
| Marks Range | GPA | Grade | Description      |
|-------------|-----|-------|------------------|
| 90–100      | 10  | O     | Outstanding      |
| 80–89       | 9   | A+    | Excellent        |
| 70–79       | 8   | A     | Very Good        |
| 60–69       | 7   | B+    | Good             |
| 50–59       | 6   | B     | Above Average    |
| 40–49       | 5   | C     | Average          |
| Below 40    | 0   | F     | Fail             |

---

## 🛠️ Installation & Usage

1.Launch the program. from code

2.Click Add Subject to add a subject entry.

3.Enter marks (0–100) for each subject.

4.Press Calculate CGPA.

5.The app displays GPA, Grade, and overall CGPA along with the grading system.
